# Ozon API Documentation

Эта папка содержит официальную документацию по API Ozon.

## Назначение

Claude будет **автоматически читать** эти файлы перед любыми изменениями в коде, связанном с Ozon API.

## Файлы для добавления

Положи сюда свои файлы с документацией:

- `ozon-seller-api.md` (или .pdf, .txt) - Seller API документация
  - Товары, остатки, заказы
  - Эндпоинты `/v2/product/info`, `/v3/product/info/stocks`, `/v2/posting/fbo/list`

- `ozon-performance-api.md` (или .pdf, .txt) - Performance API документация
  - Реклама, статистика
  - Эндпоинты `/api/client/campaign`, `/api/client/statistics`

## Как это работает

1. Ты добавляешь файлы сюда
2. В CLAUDE.md есть правило читать эти файлы перед работой с API
3. Claude автоматически проверяет документацию перед изменениями
4. Меньше ошибок, правильные эндпоинты и параметры

## Формат файлов

Любой формат: `.md`, `.pdf`, `.txt`, `.json` - Claude может читать всё.
